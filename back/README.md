# Desafio Khipo

Readme referente ao backend da aplicação

## Descrição

Nesta aplicação foram utilizados as tecnologias:

- NestJS;
- Typescript;
- Jest;
- Eslint;
- Prettier;
- Swagger.

Outra tecnologia importante que foi utilizado é o PNPM ao invés do NPM ou Yarn.

## Instalação

```bash
$ pnpm install
```

## Executando a aplicação

```bash
# development
$ pnpm run start
```

## Testes

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
